Archive manager

Welcome to use this compressed file manager. This application is developed based on an open-source compressed package management project and is open source using the MIT protocol.

Currently, this project supports various common compressed file formats, including Zip, 7z, Rar, and their sub volumes (. zxx,. 7z. xxx,. artx. rar), gz, and tar formats.

The format not mentioned may be gradually added in the future.

Due to Java's excellent compatibility, Java6 and earlier applications can run correctly in higher versions of JavaVM. Therefore, this application is developed using the Java language. If you encounter any problems during use, you can submit an issue on Github.
