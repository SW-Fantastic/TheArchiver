About Archive

Through this project, you can easily browse the content of compressed files, and its operation is very simple.

When running this project, a window (as shown in the figure above) will appear, where you can click the "Open" button (the one circled), and then you will be able to select a supported compressed file, After selecting this file and clicking Open, a new window will pop up where you can browse the content of the file.

In this window, the left side is the directory tree, and the right side is the content of the path. By clicking on the directory on the left, you can see the content contained in this location on the right. Some files can be previewed by double clicking without decompressing them.

From top to bottom, this window displays the menu bar, toolbar, main function area, and many functions of the toolbar and menu bar are consistent. You should be able to understand the functions of each option in the menu bar by name. In the toolbar, the four buttons from left to right are: open compressed file, extract selected directory and file, add new file, and delete specified file or directory from compressed file.

These functions are not always available, for example, in the case of compressed files, the editing function will be disabled because compressed files are usually large and direct editing can cause various problems.
